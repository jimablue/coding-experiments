### NextJs

Create Next.js app
`npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/master/basics/learn-starter"`

Run the development server
`cd nextjs-blog`
`npm run dev`  

starts "development server" on port 3000. 
http://localhost:3000
